Reinforcement Learning

2024-05-23

Basic Concepts：

  • State: The status of the agent with respect to the environment.
  • State Space: The set of all states \mathcal{S}=\{s_i\}_{i=1}^n
  • Action
  • Action Space of a State:
    \mathcal{A}(s_i)=\{a_i\}_{i=1}^n
  • State Transition: s_1\xrightarrow{a_2}s_2
  • Using Probability to describle State Transition, for example p(s_2|s_1,a_2)=1 (It could be stochastic)
  • Policy: Tell the agent what actions to take at a state. We can use conditional probability to represent \mathcal{P} = \pi(a_1|s_1), \pi represents a policy.
  • Reward: After taking an action, we can get an real number. Math: p(r=-1|s_1,a_1)=1 (It could be stochastic)
